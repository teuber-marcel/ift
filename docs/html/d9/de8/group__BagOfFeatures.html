typedef int(* iftBowCodingPooling) (iftMImage *mImg, float *featsOut, int n)

Coding and Pooling over iftMImage Prototype Interface to coding and pooling strategies on Bag of Features. Takes as input a multiband image, with the patches responses to each dictionary kernel (The higher the response, the closer to that kernel).

Parameters
mimgMultiband image of responses per kernel
[out]featsOutFeatures after pooling and coding
nNumber of features
Returns
The number of features (should be the same as the parameter n, only for asserting)
