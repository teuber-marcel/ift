Selects the best atlases to a test image by Normalized Mutual Information [1]. Train set and test image must be already registered.

Warning
All images should be registered in the same coordinate space.

If masks are passed for the training images and testing image, they are masked using their corresponding mask before normalized mutual information.
* The higher the NMI, more similar (better) the train image is.
[1] Aljabar, 2009 - Neuroimage - Multi-atlas based segmentation of brain images: atlas selection and its effect on accuracy

Parameters
test_imgTesting Image.
train_img_setTraining Image Set.
n_atlasesNumber of atlases to be selected.
test_img(Optional) Testing Mask.
train_mask_set(Optional) Set of masks for the training set. The pathnames must be in the same order of the training images.
selected_idxs_out(Optional) Reference to return the indices from the selected atlases.
Returns
A file set with the pathnames from the selected images.

◆ iftFindObjModelSeeds()

iftLabeledSet * iftFindObjModelSeeds ( iftObjModel *  obj_model,
iftImage *  clf_mask,
iftSet **  forbidden 
)

Finds the Object Model's Seeds for delineation. It assumes that the test image is registered on the template image (or vice-versa), used to build the object model.

The borders of the dilated and eroded certainty region of the prob. atlas form the background and object seeds, respectively. A texture classification mask/membership map can be passed to identify the regions to be adapted into the model. Thus, voxels inside the (dilated) uncertainty and certainty regions (rough segmentation) and classified as background in the classification mask are considered "forbidden for delineation".

Parameters
obj_modelObject Model.
clf_maskClassification mask (Membership Map) of the test image used to filter the seeds. If NULL, no filtering is applied.
rough_maskRough segmentation (Volume of Interest) used to define forbidden voxels inside it classified as background by the classification mask. If NULL or clf_mask is NULL, nothing is done.
forbiddenReference to save the set with the forbidden voxels identified by the texture classification mask.

Register (by Elastix) a AdaPro Model on a test image's space. Resulting registrations and mapping are assigned in the own input model.

Reference image is registerd with the test one. This will be the new reference image.
Then, all object models are mapped to new space using the deformation fields.
Deformation fields are not saved.

Parameters
adaproAdaPro Model.
test_imgTesting Image.
elastix_filesElastix Parameter Files.
def_fields_outOptional reference to return the resulting deformation fields. Pass NULL to ignore it.

Segments an Image by Classical MALF. This function considers the training label set is already registered on testing image.

Classical MALF registers all atlases (img + label img) on test image's space.
The label of each voxel from segmented image is the most frequent label (Majority Voting)
