Reads an iftCSV from file.

Author
Samuel Martins
Date
Sep 15, 2015

Reads an iftCSV from file. The CSV can be a header in the first row. However, it is only indeed considered a header, if all its columns start with letters, and at least one column from the second row is a number.

Parameters
csv_pathnameThe pathname from the CSV to be read.
Returns
The iftCSV. swig(newobject)
