iftGFLIM.h File Reference

Description

GFLIM: Graph-based Feature Learning From Image Markers – functions to learn the parameters of a sequence of convolutional layers from labeled nodes in graphs and employ the resulting model for feature extraction.

Note
Programs:
  • iftFLIM-LearnModel.c = Learns the CNN model
  • iftFLIM-ExtractFeatures.c = Extracts image features.
